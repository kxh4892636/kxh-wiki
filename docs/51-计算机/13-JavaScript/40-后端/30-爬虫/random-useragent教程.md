# random-useragent 教程

##### 安装

```typescript
pnpm add random-useragent
pnpm add --save-dev @types/random-useragent
```

##### 基础使用

```typescript
import userAgent from "random-useragent";

const agent = userAgent.getRandom((ua) => {
  return ua.osName === "Windows" && ua.osVersion === "10";
});
```
