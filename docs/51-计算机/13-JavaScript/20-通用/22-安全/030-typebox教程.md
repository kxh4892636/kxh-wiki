# typebox 教程

## 基础

##### 安装

```bash
pnpm add --save-dev @sinclair/typebox
```

##### 基本使用

```typescript
import { Type, type Static } from "@sinclair/typebox";
import { Value } from "@sinclair/typebox/value";

// 定义 schema
const T = Type.Object({
  x: Type.Number(),
  y: Type.Number(),
  z: Type.Number(),
});
// 生成对应 typescript 声明
type T = Static<typeof T>;
// 验证
Value.Check(T, value);
```

## Value

### Clone

- 深拷贝;

```typescript
const A = Value.Clone({ x: 1, y: 2, z: 3 }); // const A = { x: 1, y: 2, z: 3 }
```

### Check

- 类型验证;

```typescript
const T = Type.Object({ x: Type.Number() });
const R = Value.Check(T, { x: 1 }); // const R = true
```

### Clean

- 移除多余属性;

```typescript
const T = Type.Object({
  x: Type.Number(),
  y: Type.Number(),
});

const X = Value.Clean(T, null); // const 'X = null
const Y = Value.Clean(T, { x: 1 }); // const 'Y = { x: 1 }
const Z = Value.Clean(T, { x: 1, y: 2, z: 3 }); // const 'Z = { x: 1, y: 2 }
```

### Equal

- 深相等;

```typescript
const R = Value.Equal({ x: 1, y: 2, z: 3 }, { x: 1, y: 2, z: 3 });
```
