# 异常

## exceptions

##### traceback

- 代码错误记录.

### 常见例外

##### ZeroDivisionError

- 除法运算除数为 0.

##### FileNotFoundError

- 文件或目录不存在.

### 处理异常

##### try 语句

```python
try:
  suite
except A:
  # 异常 A
  suite
except B:
  # 异常 B
  suite
except :
  # 其余异常
  suite
else:
  suite
finally:
  suite
```

##### 跳过异常

- 在 except 中使用 pass 语句.

```python
try:
  suite
except A:
  pass
else:
  suite
finally:
  suite
```

### traceback 模块

```python
# 更详细的错误信息
import traceback

try:
    # your code here
except Exception as e:
    traceback.print_exc()
```
