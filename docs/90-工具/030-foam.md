# foam

## 安装

- vacode 拓展安装;

## 功能

##### 拓展语法

```markdown
<!-- 标签语法 -->

#tag

<!-- 内链语法 -->

[[]]
```

##### 拓展功能

- 超链接自动更新;
  - 文件重命名有效, 文件夹重命名无效;

## 命令

- Foam: Show Graph: 笔记关系图;

## 面板

- Backlinks Panel: 反向链接面板;
- Tag Explorer Panel: 标签面板;
- Orphans and Placeholder Panels: 孤儿面板;
  - 查找未被连接的笔记;
