# wsl 教程

## 基础

### 基本命令

```bash
# 安装 wsl
wsl --install
# 安装指定版本 wsl
wsl --install -d <Distribution Name>
# 列出可用的 linux 版本
wsl --list --online
# 列出已安装的 linux 发行版
wsl --list --verbose
# 更新 wsl
wsl --update
# 检查 wsl 状态
wsl --status
# 关闭 wsl
wsl --shutdown
# 卸载 wsl
wsl --unregister <DistributionName>
```

### 安装

##### 安装

```bash
wsl --install debian
```

##### 文件存储

```bash
# wsl 文件目录
\\wsl$
```

### 配置

- `win + R`，输入 .wslconfig；

```bash
[wsl2]
processors=8
memory=4GB
swap=2GB
localhostForwarding=true
```
