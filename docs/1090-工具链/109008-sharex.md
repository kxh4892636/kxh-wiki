# sharex

## 个人设置

##### 开机自启动

- 应用程序设置-综合;
- 勾选 `windows 启动时运行 sharex`.

##### 显示语言

- 应用程序设置-常规-语言;
- 选择 `简体中文`.

##### 启动时最小化到托盘

- 应用程序设置-常规;
- 勾选 `启动时最小化到托盘`.

## 录制屏幕

### 录制电脑声音

- 动作设置-截图-屏幕记录;
- 点击 `屏幕录制选项`;
- 点击 `安装"屏幕捕捉器" 和 "虚拟音频捕捉器" 视频/音频源`, 安装;
- 视频源选择 `screen-capture-recorder`;
- 音频源选择 `virtual-audio-capturer`.
